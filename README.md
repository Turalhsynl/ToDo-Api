# 📝 Personal Task Tracker API

A simple REST API for managing personal tasks. Built with Go and Gin framework. The API allows creating, reading, updating, deleting, and filtering tasks.

---

## 🚀 How to Run the Project

### 1. Clone repository

git clone https://github.com/your-username/task-tracker.git
cd task-tracker

### 2.Install dependencies
go mod tidy

### 3. Run the application
go run main.go

### Server will start on:
http://localhost:8080

⚙️ Technology Choices
Go (Gin Framework) – chosen for its simplicity, performance, and fast development of REST APIs.
SQLite  – flexible storage option for local development and production readiness.

Gin was selected because it provides lightweight routing and good performance without unnecessary complexity.
